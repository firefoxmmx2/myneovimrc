# 个人周报生成技能

## 任务

根据用户提供的工作内容和时间范围，生成个人周报Excel文件。

## 输入

用户需要提供：
1. **时间范围**：格式为 `YYYYMMDD~YYYYMMDD`，例如 `20260420~20260424`
2. **工作内容**：一周的工作记录文本

## 模板文件

技能内置模板文件：`template.xlsx`（位于技能目录）

## 处理步骤

### 1. 总结工作内容

首先使用 `/monthly-assessment` 技能对工作内容进行总结，按项目分组生成简洁的工作总结。

### 2. 解析时间范围

将时间范围 `YYYYMMDD~YYYYMMDD` 转换为：
- **文件名格式**：`个人周报-MM.DD-MM.DD.xlsx`
- **标题格式**：`个人周报（YYYY.M.D-YYYY.M.D）`

### 3. 生成Excel文件

使用内置模板文件 `template.xlsx` 创建新文件：
1. 复制模板文件到目标目录
2. 更新A1标题中的时间
3. 清空第3行及以后的数据区域
4. 填入新的总结内容

模板结构：
- **A1:D1**：合并单元格标题
- **A2-C2**：表头（工作类型、本周工作内容、下周工作计划）
- **从第3行开始**：按项目填入总结内容，每个项目占一行
- **A列**：统一填写"项目产品"

### 4. 保存文件

保存到用户指定目录（默认为当前工作目录）。

## 输出示例

**输入**：
```
20260420~20260424
山东短租房验收交付物整理打包。
行业场所数据汇聚警综涉案未成年、互联网店铺、资质数据整理。
...
```

**输出**：
- 文件名：`个人周报-04.20-04.24.xlsx`
- A1：`个人周报（2026.4.20-2026.4.24）`
- B3：`山东短租房完成验收交付物整理与前端部署恢复，实现智能检索算法优化提高准确率降低召回率，处理内网图片不显示等问题。`
- B4：`行业场所完成数据汇聚警综数据整理与同步并发度调整，实现自动刷新保存点脚本保证断点续传，处理现场环境安全报告、互联网环境安全漏洞、附件堆积拥堵等问题。`
- ...

## 注意事项

- 时间范围解析时，月份和日期要去掉前导零的显示（如04显示为4）
- 总结内容要简洁，每个项目一句话
- 保留模板的格式样式
- 不修改原模板文件，生成新文件
