# 月考核总结生成技能

## 任务

根据用户提供的一个月周报信息，生成简洁的月考核总结。

## 输入

用户输入变量：`{用户输入}` - 包含一个月周报信息的文本

## 处理步骤

### 1. 忽略日期列

提取周报内容时，忽略日期/时间列，将所有工作内容合并处理。

### 2. 按项目分组总结

将工作内容按项目分组，每个项目用**一句话**总结。

### 3. 输出格式

每个项目的总结格式：

```
[项目名称] 完成 [核心工作内容]，实现 [功能/改进]，处理 [问题/修复] 等问题。
```

**参考示例**：
```
行业场所完成 redis 集群配置与旅馆企业端服务改造，实现前端异常提示与网约房符号过滤，处理统计异常、系统卡顿等问题。
西安应急完成大屏接口免认证调用与前端 https 工作台调整，处理演示、图片异常、登录跳转等问题并修复安全漏洞、协助解决 JPA 查询问题。
```

### 4. 文本压缩

- 在不损失关键内容的前提下压缩文本
- 使用简洁的动词（完成、实现、处理、修复、协助等）
- 合并相似内容，避免重复

## 输出要求

- 每个项目独立一行
- 保持专业、简洁的语言风格
- 突出核心成果和解决的问题
